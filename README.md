# 🤖 ChatMind AI — Intelligent Conversational Assistant

🚀 A modern AI-powered chatbot built using Streamlit + OpenAI API with memory, voice input, and PDF understanding capabilities.

---

## 🌐 Live Demo

👉 Try the app here:  
🔗 **Streamlit App:** https://chatmind-ai.streamlit.app/ 

---

## ✨ Key Features

💬 **Smart AI Chatbot**  
Natural, human-like conversations powered by OpenAI.

🧠 **Memory System**  
Remembers chat history for continuous conversations.

🎨 **Multiple Themes**  
Switch between light, dark, and custom UI themes.

🎤 **Voice Input Support**  
Speak directly to the chatbot using microphone input.

📄 **PDF Intelligence**  
Upload PDFs and get instant AI summaries.

➕ **Modern ChatGPT-like UI**  
Floating “New Chat” button + clean sidebar design.

🗑️ **Clear Chat Option**  
Reset conversations anytime.

---

## 🧠 How It Works

1. User sends a message (text/voice)
2. Input is processed in Streamlit backend
3. OpenAI API generates response
4. Chat history is saved locally (`chat_history.json`)
5. UI updates instantly like ChatGPT

---

## 🛠️ Tech Stack

- 🐍 Python 3.10+
- 🎈 Streamlit
- 🤖 OpenAI API
- 🎤 SpeechRecognition + streamlit-mic-recorder
- 📄 PyPDF2
- 💾 JSON-based memory system
